# 📱 Frontend - App Android Reporte de Accidentes

Aplicación Android desarrollada en Kotlin para reportar accidentes desde dispositivos móviles.

---

## 🛠️ Tecnologías

- Android Studio
- Kotlin
- Android SDK
- Jetpack Compose
- Arquitectura MVVM / Clean Architecture
- Hilt (inyección de dependencias)
- Retrofit (consumo API)
- Room (almacenamiento local)
- Firebase App Distribution
- Emulador / Dispositivo físico

---

## 🧱 Arquitectura

- **Presentation:** Jetpack Compose + ViewModels
- **Domain:** Casos de uso
- **Data:** Repositorios, API, Room

---

## 📦 Funcionalidades

- Registro de usuario
- Inicio de sesión
- Crear reporte de accidente
- Adjuntar imágenes
- Guardar reportes localmente
- Sincronizar con backend
- Visualizar historial de reportes

---

## 🔌 Comunicación con Backend

- Retrofit
- JSON
- Autenticación con JWT

---

## 🚀 Distribución

- APK generado localmente
- Firebase App Distribution

---

## 🧪 Pruebas

- Unitarias en ViewModels
- Pruebas manuales en UI
