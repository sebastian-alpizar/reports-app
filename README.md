# 📱 Sistema de Reporte de Accidentes

Este proyecto consiste en una aplicación móvil Android y un backend REST para el registro y gestión de reportes de accidentes.

---

## 🧩 Arquitectura General

- **Frontend:** Android (Kotlin + Jetpack Compose)
- **Backend:** API REST (Kotlin + Spring Boot)
- **Base de Datos:** PostgreSQL
- **Autenticación:** OAuth2 + JWT
- **Distribución:** Firebase App Distribution
- **Contenedores:** Docker
- **Despliegue Backend:** Heroku
